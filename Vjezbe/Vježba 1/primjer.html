Frontend web tehnologije

Prvo, moramo naučiti neka osnovna HTML pravila.

Naslovi

HTML <h1> - <h6> elementi predstavljaju šest nivoa naslova odjeljaka. <h1> je najviši nivo odjeljka, a <h6> je najniži.

Liste

Na ovaj način moguće je napraviti nesređenu listu:

  • Dodajte element "ul" (označava nesređeni popis)
  • Dodajte svaku stavku u svoj vlastiti element "li"
  • Ne trebaju biti u određenom redoslijedu

Ovako izgleda uređena lista:

  1. Primijetite da novi element "ol" obavija sve
  2. Ali, tipovi elemenata liste su isti
  3. Takođe imajte na umu kako se brojevi sami povećavaju
  4. Trebali biste primijetiti da je stvar u ovom preciznom redoslijedu, jer je ovo uređena lista

Tagovi za uređivanje

Ponekad trebate skrenuti pažnju na određenu riječ ili frazu.

Drugi puta morate snažno naglasiti važnost riječi ili fraze.

Ukoliko želite da pređete u novi red, ovo nije dovoljno.

To je moguće uraditi,
ukoliko se koristi odgovarajući <br> tag.

Horizontalno pravilo

Na ovaj način moguće je postići tematsko odvajanje teksta

Također, na ovaj način moguće je promijeniti boju teksta, ali je to moguće uraditi i u eksternom css file u, kao što je urađeno za naslov dokumenta.
